## 🚀 О проекте

SportPredict.AI — это автоматизированная система прогнозов на спорт, использующая нейросеть GigaChat для анализа матчей и выбора самых перспективных ставок.

### ✨ Особенности

- 🤖 **Анализ через GigaChat** — нейросеть анализирует каждый матч
- 🔒 **Железные прогнозы** — матчи с вероятностью 90%+
- 📊 **Реальная статистика** — точность и профит в реальном времени
- 📰 **Автоматические новости** — парсинг со спортивных сайтов
- 🎨 **Красивый дизайн** — минималистичный, но богатый
- ⚡ **Автообновление** — GitHub Actions каждые 3-6 часов
